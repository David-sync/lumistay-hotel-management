# Prompt dùng với repo ui-ux-pro-max-skill

Dán prompt này cho AI trong Cursor/Claude Code sau khi đã cài skill:

```txt
Generate a polished UI/UX design system for my hotel management admin dashboard.

Project context:
- Coursework database project for a mini hotel management system.
- Stack: Next.js App Router, TypeScript, Tailwind CSS, SQL Server.
- Database already has tables, views, stored procedures, triggers and functions.
- Frontend should be an internal hotel operation console, not a public booking site.

Design direction:
- Dark luxury hotel dashboard.
- Glassmorphism cards, bento dashboard, elegant typography.
- Accent color: champagne gold.
- Secondary accent: cyan/blue for live operational statuses.
- Must look premium but still data-dense and readable.

Pages:
- Login
- Dashboard
- Rooms
- Bookings
- Customers
- Invoices
- Services
- Staff
- Reports

UX requirements:
- Sidebar navigation.
- KPI cards.
- Room status grid.
- Booking table with statuses.
- Invoice table with payment badges.
- Revenue chart section.
- Use clear empty states, loading states, error states, and toast messages.

Please suggest improvements to the current LumiStay design system and generate component-level UI recommendations.
```
